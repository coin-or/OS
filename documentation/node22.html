GLPK

GLPK is a an open-source linear and integer-programming solver from the GNU organization. See http://www.gnu.org/software/glpk/. In order to use GLPK with OS, either execute get.AllThirdParty or connect to ThirdParty/Glpk and execute get.ThirdParty.
