The OSSolverService

The OSSolverService is a command line executable designed to pass problem instances in either OSiL, AMPL nl, or MPS format to solvers and get the optimization result back to be displayed either to standard output or a specified browser. The OSSovlerService can be used to invoke a solver locally or on a remote server. It can work either synchronously or asynchronously. At present six service methods are implemented, solve, end, retrieve, getJobID, knock and kill. These methods are explained in more detail in section 10.3.
