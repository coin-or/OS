Library Paths

After running configure as described above, on Unix systems, it will be necessary to set the environment variables LDFLIBRARYFPATH or DYLDFLIBRARYFPATH (on Mac OS X) to point to the location of the installed third party libraries in the case that the libraries are dynamic and not static libraries.
