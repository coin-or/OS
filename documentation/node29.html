Platforms

The build process described in Section 3.1 has been tested on Linux, Mac OS X, and on Windows using MINGW/MSYS and CYGWIN. The gcc/g++ and Microsoft cl compiler have been tested. A number of solvers have also been tested with the OS library. For a list of tested solvers and platforms see Table 1. More detail on the platforms listed in Table 1 is given in Table 2.
