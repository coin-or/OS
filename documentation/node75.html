Examples

The example executable files are not built by running configure and make. In order to build the examples the user must first run 

make all
in the project root directory. Running make install will place all of header files required by the examples in the directory 
{\tt  COIN-OS/include}
and all of the libraries required by the examples in the directory 
{\tt  COIN-OS/lib}
Next, the user should connect to the appropriate example directory and run make. The Makefile in each example directory is fairly simple and is designed to be easily modified by the user if necessary. Developers can use the Makefiles as a starting point for building applications that use the OS project libraries.
